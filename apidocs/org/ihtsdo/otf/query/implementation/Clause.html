Class Clause

  • java.lang.Object
    • org.ihtsdo.otf.query.implementation.Clause
  • Direct Known Subclasses:
    LeafClause, ParentClause

    public abstract class Clause
extends Object
    Statements that are used to retrieve desired components in a Query. Clauses in a Query are organized into a tree and computed using a depth-first search. The root node in a Query must be an instance of a ParentClause.
