About QS sprint2: (MMP) Query service parent

Parent project of modules that support query services.

Project Modules

This project has declared the following modules:

Name Description
QS sprint2: Query rest client Enables Representational State Transfer (REST) client processing for queries and provides examples.
QS sprint2: Query jaxb objects Java data display objects generated from a JAXB generated XSD.
QS sprint2: Query rest service Representational State Transfer (REST) server connectivity and query execution.
QS sprint2: Query implementation Query clauses used for the construction and computation of queries.
QS sprint2: Query integration tests Performs integration tests for query clauses. Provides examples of query structure to demonstrate how queries can be constructed.
QS sprint2: Query documentation Contains documentation for construction of queries and getting started guide.
